Project Overview

In this project, I analyzed the sales data of a hypothetical retail store. I designed a MySQL database, organized and set up a relational schema, imported data from CSV files into tables, performed data cleaning, and conducted various analyses to uncover insights about sales trends, customer behavior, and product performance. check out the script on my Github repository.

GOAL

The goal of this project is to utilize SQL for in-depth data analysis of the sales data to identify key trends, customer behaviors, and product performance. Often, companies have data that contain a lot of hidden information. Asking the right questions will provide relevant insights and will help in making informed business decisions.

RESULT

By using SQL codes, I was able to gain valuable insights on product popularity, monthly sales trend, regional sales performance, payment mode preference and also customer value analysis. With this information, I can make an algorithm or statistical model which can be used to make the right decisions.

PROJECT DURATION

Projects duration varies between one to two weeks. I started this project by getting an understanding of the data and plotting the database design. afterwards, i delved into creating the database using relational schemas and then importing the data. Finally, data cleaning, data exploration, analysis and documentation.

Table Creation And Data Import

I divided the dataset into four tables (sales data, customer information, item details, and shipping information) to create a relational schema, facilitating efficient data management and query execution. The data from CSV files was loaded into the corresponding tables using the SQL load data in file statement. Using this statement is more efficient than the import Wizard, it took 7 seconds as opposed to the hours it would have taken importing through the Wizard.

Product Popularity And Monthly Sales Trend

I utilized SQL join queries to aggregate store sales by product, identifying the most popular items. Additionally, I analyzed seasonality and peak periods by grouping total sales by months, thereby uncovering monthly sales trends.

Sales Analysis And Optimization

I conducted a comprehensive analysis of sales distribution across the various regions, pinpointing insights. Additionally, I identified the top-performing customers and their preferred payment methods for strategy optimization.
