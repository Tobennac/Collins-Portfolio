Project Overview

This project involves building a movie visualization and recommendation system. The goal is to clean and analyze a dataset of movies, visualize key trends and metrics, and develop a content-based filtering recommendation system. check out the script on my Github repository.

GOAL

The goal of this project is to perform data cleaning and exploratory data analysis on a movie dataset. Also to visualize key metrics and trends within the dataset and build a content-based filtering recommendation system for movies.

RESULT

  • Successfully cleaned the movie dataset, removing inconsistencies and handling missing values.
  • Generated insightful visualizations showing trends in movie ratings, genres, and other key attributes.
  • Developed a recommendation system that suggests similar movies based on user input.
